Positionnement

La technique du positionnement en CSS utilise l'attribut position. Celui-ci peut avoir l'une des quatre valeurs suivantes :

  • position: static; /* Valeur par défaut */
  • position: absolute; /* ! */
  • position: relative; /* ! */
  • position: fixed; /* ! */
  • position: sticky; /* ! */

Les repères de positionnement peuvent être combinés pour obtenir l'effet désiré.
